# Sproochmaschinn API Workshop

This repository is a Binder-ready workshop scaffold for exploring the Sproochmaschinn API with Python notebooks.

## Repository layout

- `requirements.txt` — Python dependencies for Binder
- `postBuild` — optional Binder post-build commands
- `sproochmaschinn_helpers.py` — reusable helper functions
- `data/` — sample files for workshop exercises
- `notebooks/` — workshop notebooks from basic to advanced

## Suggested flow

Core path:
1. `00_start_here.ipynb`
2. `01_tts_basics.ipynb`
3. `02_stt_basics.ipynb`
4. `03_exports_and_analysis.ipynb`

Advanced path:
5. `04_advanced_helper_functions.ipynb`
6. `05_advanced_small_client.ipynb`
7. `06_advanced_challenges.ipynb`
